Chat Analyzer (Чат талдаушы)

Сипаттамасы

Бұл жоба чаттағы хабарламаларды талдауға арналған қарапайым бағдарлама.

Бағдарлама:

- Әр қолданушы қанша хабарлама жібергенін есептейді
- Ең жиі қолданылатын сөздерді анықтайды

 Мақсаты

Java тілінде "HashMap" коллекциясын қолдану арқылы мәтіндік деректерді өңдеуді үйрену.

⚙️ Қалай жұмыс істейді

Хабарламалар келесі форматта сақталады:

username: message

Бағдарлама:

1. Әр хабарламаны қолданушы мен мәтінге бөледі
2. "HashMap" арқылы қолданушылардың хабарлама санын есептейді
3. Мәтінді сөздерге бөледі
4. Әр сөздің жиілігін есептейді

Негізгі ұғымдар

- "HashMap" (кілт–мән құрылымы)
- Мәтінді бөлу ("split()")
- Жиілікті есептеу ("getOrDefault()")

Қолдану

1. Бағдарламаны іске қосыңыз
2. "messages" тізімін өзгертіп, өз деректеріңізді енгізіңіз
3. Нәтижені консольдан көріңіз
 Қорытынды

Бұл жоба Java коллекцияларын пайдаланып, мәтіндік ақпаратты тиімді өңдеуді және статистика шығаруды көрсетеді.
